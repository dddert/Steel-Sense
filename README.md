# Steel-Sense
Мобильная CV система для подсчета металлопроката

## Требования

**Аппаратные:**
* Raspberry Pi 4 (рекомендуется 4GB+ RAM)
* Модуль камеры (поддерживается через `picamera2`)
* Сенсорный экран

**Программные:**
* Raspberry Pi OS (64-bit)
* Python 3.9+

## Установка

1. Установите системные зависимости:
```bash
sudo apt update
sudo apt install python3-pip python3-pyqt5
```

2. Установите необходимые Python-библиотеки:
```bash
pip3 install ultralytics opencv-python numpy streamlit pandas
```
*Примечание: библиотека `picamera2` предустановлена в стандартных образах Raspberry Pi OS.*

3. Загрузите обученные веса модели YOLO-seg (файл `.pt`) в директорию проекта.

## Настройка

Откройте файл `main.py` и укажите корректный путь к файлу весов в константе `MODEL_PATH`:

```python
MODEL_PATH = "/home/user/project/best.pt"
```

## Запуск

Для запуска приложения выполните команду:

```bash
python3 main.py
```

**Поведение системы при запуске:**
1. На экране Raspberry Pi развернется полноэкранный графический интерфейс с live-трансляцией с камеры.
2. В фоновом процессе автоматически инициализируется веб-сервер Streamlit.

**Доступ к веб-интерфейсу:**
1. Подключите внешний компьютер или ноутбук к той же локальной сети, что и Raspberry Pi.
2. Обратите внимание на статусную строку в нижней части экрана Raspberry Pi — там будет указан локальный IP-адрес устройства.
3. Откройте любой веб-браузер на внешнем компьютере и перейдите по адресу: `http://<IP-адрес-Raspberry-Pi>:8501`.
4. Веб-интерфейс предоставит доступ к журналу операций, оригинальным снимкам и результатам сегментации.

## Структура проекта

* `main.py` — ядро системы: графический интерфейс, управление камерой, фоновый инференс нейросети.
* `web_app.py` — веб-приложение для визуализации накопленных данных.
* `captures/` — локальное хранилище для фотографий, бинарных масок и JSON-файлов с метаданными (создается автоматически при первом сканировании).
